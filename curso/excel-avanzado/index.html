Excel Avanzado

35 horas

Curso avanzado de Microsoft Excel diseñado para profesionales. Domina herramientas de análisis de datos, tablas dinámicas, macros y funciones complejas para optimizar la toma de decisiones.

Contenido del Curso

Excel Avanzado (35 horas)

1. NIVELACIÓN Y REPASO

  • Habilidades Básicas: Resumen de Excel Elemental
  • Conceptos clave de la hoja de cálculo

2. GESTIÓN AVANZADA DE HOJAS

  • Uso de Plantillas
  • Presentación y esquematización de hojas
  • Introducción a Macros: Grabación y ejecución básica
  • Funciones avanzadas (Lógicas, Fecha, Hora, Búsqueda)

3. LIBROS DE TRABAJO

  • Presentación profesional de libros
  • Gráficos incrustados vs hojas de gráficos
  • Elaboración de auditoría de fórmulas
  • Gestión de comentarios y colaboración

4. GRÁFICOS AVANZADOS

  • Creación y personalización avanzada
  • Tipos de gráficos especiales
  • Formatos numéricos personalizados en gráficos

5. FILTROS Y BASES DE DATOS

  • Filtros y Auto filtros avanzados
  • Ordenamiento multinivel
  • Organización y resumen de datos

6. ANÁLISIS DE DATOS I

  • Tablas dinámicas: Creación y configuración
  • Recuperación de datos externos
  • Gestión de listas y tablas oficiales
  • Exportación de datos a la Web

7. ANÁLISIS DE DATOS II

  • Uso de Escenarios y Administración de escenarios
  • Manejo de variables múltiples (Buscar objetivo, Solver)
  • Uso de gráficos dinámicos para análisis
  • Funciones para análisis estadístico
