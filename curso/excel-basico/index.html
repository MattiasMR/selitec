Excel Básico

35 horas

Curso introductorio a Microsoft Excel. Aprende a crear hojas de cálculo, utilizar fórmulas básicas, diseñar gráficos y organizar datos de manera eficiente para mejorar tu productividad laboral.

Contenido del Curso

Excel Básico (35 horas)

1. INTRODUCCIÓN A EXCEL

  • Elementos de la pantalla de trabajo (Interfaz)
  • Barras de herramientas y menús
  • Técnicas básicas de una hoja de cálculo
  • Selección de celdas e inserción de datos
  • Funciones básicas

2. OPERACIONES Y FUNCIONES

  • Introducción de fórmulas
  • Direcciones de celdas: relativas, absolutas y mixtas
  • Funciones Matemáticas
  • Funciones Estadísticas Básicas
  • Introducción a funciones Lógicas y de Búsqueda

3. CREACIÓN DE GRÁFICOS

  • Crear gráficos a partir de datos
  • Modificar tipos y estilos de gráficos
  • Personalización y elementos del gráfico
  • Impresión de gráficos

4. FORMATO DE HOJA DE CÁLCULO

  • Formatos de celda (Bordes, Relleno, Fuente)
  • Formatos numéricos (Moneda, Fecha, Porcentaje)
  • Alineaciones y combinación de celdas
  • Inserción de imágenes y formas

5. GESTIÓN DE BASES DE DATOS SIMPLES

  • Creación de una lista de datos
  • Ordenar datos (alfabético, numérico)
  • Uso de Filtros básicos
  • Resultados y Sub-totales simples

6. CONFIGURACIÓN E IMPRESIÓN

  • Configuración de página (márgenes, orientación, tamaño)
  • Encabezados y pies de página
  • Insertar número de páginas
  • Áreas de impresión y saltos de página
